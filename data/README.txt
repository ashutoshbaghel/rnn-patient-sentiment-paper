This is a corpus of doctor reviews scraped from RateMD.com. Each file is a particular state and first letter of last name (this is due to the structure of RateMD). The data format should be self-explanatory (it is tab-delimited). If you use this data, please cite:

What Affects Patient (Dis)satisfaction? Analyzing Online Doctor Ratings with a Joint Topic-Sentiment Model
Michael J. Paul, Byron C. Wallace and Mark Dredze
AAAI Workshop on Expanding the Boundaries of Health Informatics Using AI (HIAI), 2013
